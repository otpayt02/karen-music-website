    App Guide Shortcuts, workflows, rules, and sequences
    Start Here

    The editor listens for chart shortcuts only when focus is on the invisible editor target. When you close the sidebar, exit the wizard, or return from the Chart tab, focus is routed there automatically.

    ?
    Open or close this guide.
    ☰ / ×
    Open or close the sidebar. Closing the sidebar returns focus to the chart editor.
    Gear
    Open settings for language, theme, UI design direction, palette, motion, density, time signature, default measures per row, Karen keyboard immersion, and alternate song folder.
    Chart / Lyrics
    Use the paper flaps above the sheet to bring either the chord chart or the lyrics sheet to the front.
    Core Chart Navigation
    Arrow keys
    Move the active beat left, right, up, or down. Right Arrow creates a new measure when you move past the last beat of the section.
    Space
    Move forward to the next beat. At the final beat of the section, create a new measure and land on beat 1.
    Tab
    Insert a new full measure directly after the current measure, shift later measures right, and land on beat 1 of the new measure.
    Ctrl + Space
    Insert a new measure directly after the current one using the current beat number as the new measure length.
    Ctrl + Tab
    Split the current measure after the active beat, then move into a new measure.
    Backspace
    Smart delete: remove the latest visible change from the beat, or move backward/delete empty trailing structures when the current beat is empty.
    Delete
    Delete the current measure. If the section has only one empty measure, delete the section and jump back.
    Esc
    Clear pending states such as bass-entry or triangle/superscript mode.
    Chord Entry Rules
    A-G
    Set the chord root on the active beat. B creates a B root when the beat is empty.
    B
    When a root already exists, toggle flat on that chord.
    M
    Toggle minor on the active chord.
    7
    Add or toggle a plain seventh.
    T then 7
    Enter the triangle/superscript seventh sequence.
    / then A-G
    Arm bass mode, then press a note letter to set the bass note. Press / again on a beat with bass to clear it.

    Most chord-quality commands require a root chord first. X marks are allowed on empty beats so you can mark rhythm before choosing a chord.

    Rhythm Marks And Spans
    .
    Toggle push: dot plus underline behavior on the active rooted beat.
    X
    Add x marks for rhythmic hits. It cycles up to four manual x marks.
    O
    Toggle a circle. Turning on a circle clears dot/x marks for that beat.
    R
    Toggle a roll for the current measure. Measure-level roll clears beat-level roll marks.
    S
    Start, finish, or delete a beat-spanning slur.
    ,
    Start or finish an Only / All In cue span.
    Ctrl + 1-4
    Toggle manual underline on beat 1, 2, 3, or 4 of the current measure when the target beat has no structural underline.
    Sections And Measures
    Enter
    Open the Add Section modal from the editor.
    I / V / P / C / B / S / E
    Inside the section modal, add Intro, Verse, Pre-Chorus, Chorus, Bridge, Solo, or Ending.
    Esc
    Close the section modal without adding a section.
    + between sections
    Click an insert line to add a new section at that location.
    Left section rail
    Toggle a 2-beat pre-measure for that section.
    Row +/-
    Use the section controls to change how many measures fit on each row.
    Lead Melody Modal
    L
    Open the lead melody modal for the current measure.
    Tab 1: 1-8
    Highlight the base octave. Press Enter to confirm and move to lead spacing.
    Tab 2: 3 / 4
    Choose lead spaces per beat.
    Tab 2: ← / →
    Switch between 3-slot and 4-slot lead spacing.
    Tab 2/3: ↑ / ↓
    Shift the working octave wheel up or down.
    Tab 3: 1-7
    Place a scale-degree note in the active lead cell.
    Tab 3: 0 / Backspace / Delete
    Clear the active lead cell.
    Tab 3: ← / → / Space
    Move across the lead grid.
    Enter
    Advance to the next lead step, or save when the modal is complete.
    Ctrl + Z / Y
    Undo or redo edits inside the lead modal.
    Esc
    Cancel and close the lead modal.
    Lyrics Sheet
    1. Click the Lyrics flap above the paper stack.
    2. Add lyrics sections directly on the lyrics sheet.
    3. Use the section toggles for romanization and optional English translation.
    4. Click Chart to return to chord editing; focus returns to the editor target.

    The Notes field is separate from lyrics. Lyrics are edited on the lyrics sheet.

    Karen Keyboard Modal
    Esc
    Close the immersive Karen keyboard.
    ×
    Close the immersive Karen keyboard from the top-right corner.
    Backspace
    Undo the latest Karen keyboard insertion or delete the previous character when there is no keyboard insertion history.
    Ctrl + Z
    Undo the latest Karen keyboard insertion while the modal is open, or use the normal field undo when no keyboard insertion is pending.
    Ctrl + Y
    Redo the latest Karen keyboard insertion while the modal is open.
    Sidebar Workflows
    New
    Clear the current form and open the new-song wizard.
    Save
    Save the song metadata, chart JSON, lead rows, dates, notes, and reference media.
    Print
    Create one print packet for each selected instrument plus a Reference copy.
    Export
    Download the local song database.
    Instruments
    Select instruments in the wheel/list. Selected instruments control print packets and footer labels.
    Dates
    Date Created, Next Performance, and performed-date log flow into the sheet footer and saved song metadata.
    Reference / Media
    Store YouTube links, handwritten/photo links, and notes for the song.
    Settings And Appearance
    Wizard mode
    Choose quick song setup or guided step-by-step setup.
    Language
    Switch between English and Karen UI labels.
    Theme
    Use Auto, Light, or Dark modes when available.
    Palette
    Change the interface color set without changing chart data.
    UI Design
    Switch among Desk, Console, and Stage layouts from the same shared app logic.
    Motion
    Choose silky or calmer motion.
    Density
    Choose compact or comfy control spacing.
    Karen Keyboard
    Turn the immersive floating keyboard animation on or off. The same Karen text fields stay touch-ready either way.
    Time Signature
    Switch between 4/4 and 3/4 for new measures. Existing measures keep their saved beat counts so switching does not delete beat data.
    Measures / row
    Set the default row density for new/auto sections.
    Alternate folder
    Choose or save a different song folder path for the backend settings.
    Undo, Print, And Safety
    Ctrl + Z
    Undo the latest chart edit.
    Ctrl + Y
    Redo the latest undone chart edit.
    Ctrl + P
    Print instrument packets.
    Browser focus
    If shortcuts stop responding, close text fields/sidebar or click the chart paper; the editor target will receive shortcut focus again.
